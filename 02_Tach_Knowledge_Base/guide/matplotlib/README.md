# Guía de Matplotlib

Bienvenido a la guía de estudio para **Matplotlib**. Esta carpeta contiene recursos paso a paso para dominar la visualización de datos en Python.

## Contenido

| Archivo | Descripción |
| :--- | :--- |
| **[00_intro.md](./00_intro.md)** | Introducción, instalación y la anatomía de una Figura y Ejes. |
| **[01_basic_plots.md](./01_basic_plots.md)** | Gráficos esenciales: Línea, Dispersión, Barras, Histograma y Boxplots. |
| **[02_customization.md](./02_customization.md)** | Personalización de gráficos: Títulos, Leyendas, Colores y Estilos. |
| **[03_styling_and_aesthetics.md](./03_styling_and_aesthetics.md)** | Estilo y Estética: Uso de plt.style, Seaborn y configuraciones globales para gráficos profesionales. |

## Cómo usar esta guía

Se recomienda seguir los archivos en orden numérico. Cada archivo contiene fragmentos de código que puedes copiar y ejecutar en tu entorno local o en un Jupyter Notebook.
